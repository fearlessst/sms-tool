# sms-tool
 TR `Türkçe` 
 
`-` Sms tool günde 1 kere ücretsiz sms atmanızı sağlar (ip değişirseniz veya vpn kullanırsanız artmaktadır)

Kullanımı kolaydır 

python3 sms-tool.py
